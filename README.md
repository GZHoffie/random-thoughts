# 想到啥写啥

- 世上无难事，只要肯放弃。
- 大家都说“听不见中提琴的声音”，确实如此。我尝试中提琴声部之后，却发现这可能是最好的声部：因为自己都听不见自己，反而能听清楚整个乐团的和声。
- 家人投资基金，买了就亏，卖了就涨，大叹“亏了”、“不该卖”。我说要这么想：**正是因为**你卖了，**所以**人家才能涨。这么一想，就释然了。
- 船到桥头自然沉。
- 所谓道德，也许其根本是源于对不道德的厌恶。没有道德固然不行。但若喜欢站在道德的制高点上，周围的便会充斥着不道德，进而引发对周围一切的厌恶。这是否又是另一种不道德呢。
- 大家总说古典音乐是“高雅艺术”，去听古典音乐会更是“高贵”。但是现在去听一场古典音乐会，甚至不用花一张流行音乐会门票的十分之一。更何况，每场演出都能听到上百人的绝妙配合。这性价比已经是不能更“大众”了！
- 在有些领域，论文每次被当作baseline引用，就是被新方法打败了一次。所以引用越少，论文越强（狗头）。
- 读博对我来说最宝贵的，一是知道了我上限在哪，二是知道了我到底喜欢什么。知道上限在哪，就不用费力去追求超出自己上限的东西；知道自己喜欢什么，就学会了取悦自己，生活就有了依靠、有了盼头。
