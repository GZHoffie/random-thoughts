# 想到啥写啥

- 大家都说“听不见中提琴的声音”。确实如此。我尝试中提琴声部之后，却发现这可能是最好的声部：因为自己都听不见自己，反而能听到整个乐团的和声。
- 所谓道德，也许其根本是源于对不道德的厌恶。没有道德固然不行。但若总是喜欢站在道德的制高点上，周围的低地便会充斥着不道德，进而引发对周围一切的厌恶。对周围的美好自然就会选择无视。“知小礼而无大义，拘小节而无大德”，或许是源于此。
- 人们总说古典音乐是“高雅艺术”，去听古典音乐会更是“高贵”。但是现在去听一场古典音乐会，甚至不用花一张流行音乐会门票的十分之一。更何况，每场演出都能听到上百人的绝妙配合。这性价比已经是不能更“大众”了！
- 在有些领域，论文每次被当作baseline引用，就是被新方法打败了一次。所以引用越少，论文越强:-)。
